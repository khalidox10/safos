## 📊 أقسام لوحة التحكم

من لوحة التحكم (`/#admin`) يمكنكِ التحكم في:

| التبويب | ما يمكن تعديله |
|--------|----------------|
| **📊 لوحة القيادة** | نظرة عامة، إحصائيات، طلبيات حديثة |
| **🛍️ الطلبيات** | جميع الطلبيات، تغيير الحالة، تواصل |
| **🏪 المتجر والبراند** | الاسم، الشعار، الألوان، النصوص، الصور |
| **📦 المنتجات** | إضافة/تعديل/حذف منتجات، أسعار، مخزون |
| **📁 المجموعات** | مجموعات المنتجات، صور، أوصاف |
| **💬 آراء العملاء** | شهادات العميلات، تقييمات |
| **✂️ مراحل الصنع** | خطوات التصنيع، صور، أوصاف |
| **⚙️ الإعدادات** | تغيير كلمة المرور، تصدير/استيراد |

## 🚀 النشر

### للزبائن:
1. اضغطي **✅ نشر** في شريط الإدارة
2. شغّلي `npm run build`
3. ارفعي مجلد `dist/` للاستضافة
4. الزبائن يرون: `https://safos.online`

### لكِ (تعديل):
- افتحي: `https://safos.online/#admin`
- أو: `https://safos.online/preview.html`

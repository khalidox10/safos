## 🔐 الدخول للوحة التحكم

### الطريقة 1: شريط الإدارة
1. افتحي الموقع من متصفحك
2. اضغطي على **🔒 لوحة التحكم** في الشريط العلوي
3. أدخلي كلمة المرور: `safos1007`

### الطريقة 2: الرابط المباشر
```
https://safos.online/#admin
```

### الطريقة 3: Footer
في أسفل الصفحة → "خدمة العملاء" → "لوحة التحكم"

### 🔐 تغيير كلمة المرور
بعد تسجيل الدخول:
1. اذهبي إلى تبويب **⚙️ الإعدادات** في القائمة الجانبية
2. أدخلي كلمة المرور الحالية
3. اختاري كلمة مرور جديدة (6 أحرف على الأقل)
4. أكّدي كلمة المرور الجديدة
5. اضغطي **حفظ**

> ⚠️ **مهم**: احفظي كلمة المرور الجديدة في مكان آمن! لا يمكن استرجاعها في حال نسيانها.

## 📦 النسخ الاحتياطي والاستعادة

### تصدير البيانات (Backup)
1. اضغطي على **📥 تصدير** في شريط الإدارة
2. سيتم تحميل ملف JSON يحتوي على:
   - جميع المنتجات
   - المعلومات والمتجر
   - الطلبيات
   - حالة النشر

### استيراد البيانات (Restore)
1. اضغطي على **📤 استيراد** في شريط الإدارة
2. الصقي محتوى ملف JSON أو ارفعيه
3. اضغطي "استيراد"
4. ⚠️ **تحذير**: سيُستبدل كل شيء بالبيانات المستوردة!

## 🌐 نشر الموقع على safos.online

### الخطوة 1: بناء النسخة النهائية
```bash
npm run build
```
سيتم إنشاء مجلد `dist/` يحتوي على الموقع الكامل.

### الخطوة 2: رفع الملفات للاستضافة

#### خيار A: Vercel (موصى به)
```bash
npm install -g vercel
vercel --prod
```
ثم في إعدادات Vercel:
- أضيفي الدومين: `safos.online`
- اضبطي DNS حسب تعليمات Vercel

#### خيار B: Netlify
1. ارفعي مجلد `dist/` على Netlify Drop
2. في Domain Settings أضيفي `safos.online`
3. اضبطي DNS:
   ```
   Type: A    Name: @    Value: 75.2.60.5
   Type: CNAME Name: www Value: safos-online.netlify.app
   ```

#### خيار C: استضافة تقليدية (cPanel)
1. ارفعي محتويات مجلد `dist/` عبر FTP
2. في إعدادات الدومين أضيفي:
   ```
   Document Root: /public_html/safos.online
   ```

### الخطوة 3: إعداد DNS
في موفر الدومين (GoDaddy, Namecheap, إلخ):
```
Type    Name    Value                   TTL
A       @       [IP الاستضافة]          Auto
CNAME   www     safos.online.           Auto
```

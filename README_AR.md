## 🚀 البدء السريع

### 1. تشغيل محلياً
```bash
npm install
npm run dev
```

### 2. فتح لوحة التحكم
- افتحي `http://localhost:5173`
- اضغطي ⚙️ أو 🔒 في الشريط العلوي
- كلمة المرور الافتراضية: `safos1007`
- ⚠️ **غيّريها من**: لوحة التحكم → ⚙️ الإعدادات

### 3. التعديل والنشر
1. عدّلي ما تريدين في لوحة التحكم
2. جميع التغييرات تُحفظ كـ **مسودة**
3. راجعي في المعاينة
4. اضغطي **✅ نشر** ليظهر للزبائن

# 🛍️ LUMIÈRE - متجر حقائب فاخرة

موقع احترافي متكامل لعلامة تجارية فاخرة للحقائب النسائية المصنوعة يدوياً، مع لوحة تحكم كاملة لإدارة المتجر.

## ✨ المميزات الرئيسية

### 🎨 التصميم
- تصميم عصري وأنيق بألوان فاخرة (ذهبي، كريمي، بني إسبرسو)
- متجاوب بالكامل (Responsive) مع جميع الأجهزة
- دعم كامل للغة العربية مع RTL
- خطوط احترافية (Playfair Display + Cairo)
- تأثيرات حركية سلسة باستخدام Framer Motion
- صور حقيقية عالية الجودة من Pexels

### 🏪 أقسام الموقع
1. **شريط إعلاني متحرك** - عروض ومعلومات الشحن
2. **Navbar متطور** - قائمة تنقل ذكية مع شريط بحث وأيقونات
3. **Hero Section** - عرض رئيسي مع parallax scrolling
4. **شريط الماركيز** - أسماء مجلات الأزياء العالمية
5. **المميزات** - شحن مجاني، ضمان، صناعة يدوية
6. **المجموعات** - 3 مجموعات (يومية، سهرات، سفر)
7. **المتجر** - عرض المنتجات مع فلاتر
8. **قصتنا** - تاريخ البراند والحرفية
9. **مراحل الصنع** - 4 مراحل تفصيلية
10. **آراء العملاء** - شهادات مع carousel تلقائي
11. **معرض Instagram** - صور تفاعلية
12. **Newsletter** - نموذج اشتراك
13. **Footer كامل** - معلومات التواصل وطرق الدفع

### 🛒 الوظائف التفاعلية
- سلة تسوق جانبية مع إدارة كاملة
- فلترة المنتجات حسب الفئة
- إشعارات Toast عند الإضافة
- زر WhatsApp عائم للتواصل
- Smooth scroll بين الأقسام
- حفظ البيانات في localStorage

## 🔐 لوحة التحكم (Admin Dashboard)

### كيفية الوصول للوحة التحكم

**الطريقة 1: زر Settings في Navbar**
- اضغطي على أيقونة ⚙️ في الأعلى

**الطريقة 2: رابط في Footer**
- في قسم "خدمة العملاء" → "لوحة التحكم"

**الطريقة 3: رابط مباشر**
- أضيفي `#admin` في نهاية الرابط: `your-site.com/#admin`

**الطريقة 4: من القائمة الجانبية (Mobile)**
- افتحي القائمة → "لوحة التحكم"

### تسجيل الدخول

```
كلمة المرور: admin123
أو: lumiere2026
```

### ما يمكنكِ تعديله

#### 📊 Dashboard (الرئيسية)
- نظرة عامة على الإحصائيات
- نصائح سريعة
- معلومات التواصل الحالية

#### 🏪 معلومات المتجر
- **معلومات البراند**: الاسم، الشعار الفرعي
- **قسم البطل (Hero)**: العنوان، النص الفرعي، الوصف، الصورة
- **قصتنا**: العنوان، الوصف، سنة التأسيس، المدينة
- **العملة**: الرمز (د.م)، كود العملة (MAD)
- **الشريط العلوي**: نص الإعلانات
- **معلومات التواصل**: الهاتف، البريد، العنوان

#### 📦 المنتجات
- **عرض جميع المنتجات** مع الصور والأسعار
- **إضافة منتج جديد**: اسم، سعر، صورة، لون، فئة، وسم
- **تعديل منتج**: أي حقل قابل للتعديل
- **حذف منتج**: مع تأكيد قبل الحذف
- **الفئات**: توت، كتف، كلتش، كروس، باكت، ساتشل

#### 📁 المجموعات
- تعديل العنوان (عربي وإنجليزي)
- تغيير الصورة
- تحديث عدد القطع

#### 💬 آراء العملاء
- تعديل الاسم والمدينة
- تغيير النص
- تحديث التقييم (1-5 نجوم)
- تغيير الحرف الأول (Avatar)

#### ✂️ مراحل الصنع
- تعديل الرقم والعنوان
- تغيير الوصف
- تحديث الصورة

### 💾 الحفظ التلقائي
- جميع التعديلات تُحفظ تلقائياً في المتصفح (localStorage)
- لا حاجة لإعادة التحميل
- التغييرات تظهر فوراً في الموقع

### 🔄 إعادة التعيين
- من الشريط الجانبي → "إعادة تعيين الكل"
- سيتم استعادة جميع البيانات الافتراضية
- يتطلب تأكيد قبل التنفيذ

## 💰 العملة

العملة الافتراضية: **الدرهم المغربي (د.م)**

### تغيير العملة
1. ادخلي لوحة التحكم
2. انتقلي إلى "معلومات المتجر"
3. في قسم "العملة والإعدادات":
   - **رمز العملة**: مثلاً `ر.س` أو `$` أو `€`
   - **كود العملة**: مثلاً `SAR` أو `USD` أو `EUR`
4. اضغطي "حفظ التغييرات"

## 📝 ملاحظات مهمة

### البيانات
- جميع البيانات تُحفظ محلياً في المتصفح
- إذا مسحتِ الكاش، ستعود البيانات الافتراضية
- للتصدير/الاستيراد، يمكنكِ نسخ البيانات من localStorage

### الصور
- الصور الحالية من Pexels (مجانية)
- يمكنكِ استبدالها بصور منتجاتكِ الحقيقية
- استخدمي روابط الصور من CDN أو رفعها على خدمة استضافة

### الأداء
- الموقع محسّن للسرعة
- الصور محسّنة تلقائياً
- الكود مقسم إلى chunks للتحميل الأسرع
